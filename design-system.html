Motion

All animations are gated behind @media (prefers-reduced-motion: no-preference). Two durations, three easing curves.

Token Value Usage Demo
--duration-fast 150ms Color, opacity transitions
--duration-base 250ms Translate, shadow, border
--ease-out cubic-bezier(0.16,1,0.3,1) Entrance, fade-in —
--ease-spring cubic-bezier(0.34,1.56,0.64,1) Button lift, card hover —

Scroll reveal uses IntersectionObserver: elements get .reveal in HTML, .visible is added when they enter the viewport. Opacity 0→1 + translateY 20px→0.

.btn--primary {
  background-color: var(--color-accent-primary);
  color: #fff;
}
.btn--primary:hover {
  background-color: #5f88ff;
  box-shadow: 0 8px 28px var(--color-accent-primary-glow);

/* Lift animation (motion-safe only) */
@media (prefers-reduced-motion: no-preference) {
  .btn:hover  { translate: 0 -2px; }
  .btn:active { translate: 0  0;   }
}
